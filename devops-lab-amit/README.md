## 🛠️ Step-by-Step Practical Testing & Workflows

### 1. Docker Foundations
- Build customized images: `docker build -t amit-node-app:v1 ./docker`
- Run detached workloads: `docker run -d -p 3000:3000 amit-node-app:v1`

### 2. Multi-Tier Microservices Orchestration
- Launch orchestrated web server and database tiers: `docker compose -f compose/wordpress.yml up -d`

### 3. Maven Build & Package Lifecycle
- Clean caches, resolve coordinates, and build artifacts: `mvn clean package`

### 4. GitHub Actions Continuous Integration
- Push your main commits to `origin` to kickstart automated test executions.

### 5. Automated Declarative Pipeline Tests
- Add a standalone `Jenkinsfile` at your project root directory.
