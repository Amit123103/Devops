Unit 0: Complete Setup & Practice Guide

Your blueprint to completely install the required developer environment and master every DevOps tool in real time.

Environment Setup Test - Animated Simulation
📖 Detailed Step-by-Step Guidance (Prerequisites Setup)
Step 1: Container Tools Installation

Go to the official Docker Desktop website. Download and install Docker for your specific operating system (Windows/macOS/Linux). Ensure virtualization is enabled in BIOS.

Step 2: Install Java Development Kit & Maven

Install **OpenJDK 17**. Download **Apache Maven** and add the bin/ folder to your system environment variables (PATH). Verify by running mvn -v in the terminal.

Step 3: Setup Git & Repository Controls

Download **Git SCM**. Initialize your GitHub profile and test synchronization using terminal pushes. Create directories for all 6 DevOps units locally to stay organized.

Step 4: Continuous Integration Engine Setup

Run your standalone **Jenkins** server using Docker: docker run -d -p 8080:8080 -v jenkins_home:/var/jenkins_home jenkins/jenkins:lts.

Docker Live Execution - Animated Simulation
📖 Complete Step-by-Step Guidance (Unit 1)
Step 1.1: Environment Setup

Log in to your local workspace. Ensure Docker daemon is operational by executing the diagnostic command docker version.

Step 1.2: Image Acquisition

Pull an isolated Nginx base from remote registries to the host by running: docker pull nginx:alpine.

Step 1.3: Container Invocation

Spin up the container in isolated background mode using port forwarding flags: docker run -d --name amit_web_server -p 8080:80 nginx:alpine.
